Methode wird aufgerufen, wenn eine Nachricht im Chat-room geschrieben wurde. Reagiert auf Chat-Nachrichten: topic: Ändert das Topic. nick: Ändert den Nickname. quit: Verlässt den IRC-Chat. mode: Ändert den Channel mode (z.B.: +t = geschütztes Topic) Botname: logging on: schaltet das logging ein. Botname: logging off: schatet das Logging aus. Ruft die Logging-Funktion auf.
